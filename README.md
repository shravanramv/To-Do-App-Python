# To-Do App

This is a basic To-Do web application built entirely using the Flet library in Python.

## Run Locally

Clone the project

```bash
  git clone https://github.com/shravanramv/Python-ToDo-App.git
```

Install dependencies

```bash
  pip install flet
```

Run the web app locally

```bash
  python main.py
```
